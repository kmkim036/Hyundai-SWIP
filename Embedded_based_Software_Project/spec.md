# Hardware
## Input
- SW 1: GPIO (Polling, Interrupt)  
- SW 2: GPIO (Polling, Interrupt)  
- Potentiometer: ADC (Polling)  
- Light Sensor: ADC (Polling)  
- LM35(Temperature): ADC (Polling)
- Ultrasonic Sensor: GPIO  

## Output
- LED Red: GPIO, PWM  
- LED Blue: GPIO, PWM  
- RGB LED: GPIO(ADC), PWM   
- Buzzer: PWM
